## Overview
Burgers is a full stack application using REACT, Bootstrap in the frontend; Ruby on Rails backend, Postgress database

## Technologies
Backend API: Ruby on Rails<br/>
Database: Postgres<br/>
CRUD Route Model: Burger<br/>
Deployed on Heroku and Netlify<br/>
Frontend Framework: React, Bootstrap<br/>
Authorization: Auth0<br/>


## Backend
 Backend language: Ruby 3.0.3<br/>
 Postgres Database: Models "burger" and "user"<br/>
 Heroku backend deployment: https://burger-be.herokuapp.com/burgers/<br/>
 github backend: https://github.com/aip956/Burger_BE<br/>
